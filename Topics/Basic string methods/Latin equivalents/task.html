Latin equivalents

Imagine you have a text-processing program that doesn't work with the ASCII characters. Still, you need to use it and don't have time to correct that issue. You've decided to write another program that will replace all the letters with ligatures and diacritic marks with their equivalents from the Latin alphabet. The list of replacements is the following:

é -> e
ë -> e
á -> a
å -> aa
œ -> oe
æ -> ae

Complete the code below so that the function normalize (def is a keyword used to declare functions) would take a string with diacritics and return the one where all of them have been replaced with the equivalents.


Sample Input:
Charlotte Brontë

Sample Output:
Charlotte Bronte



Memory limit: 256 MB
Time limit: 15 seconds
